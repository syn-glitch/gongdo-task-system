# 공도 업무 관리 시스템 - 대시보드 구축 가이드

이 문서는 사용자가 직접 데이터나 표를 조작할 필요 없이 단 한 번의 **[클릭]** 만으로 실시간 **업무 현황 대시보드** 탭을 자동 생성하는 방법을 안내합니다.

---

## 🚀 대시보드 원클릭 자동 생성 방법

이제 피벗 테이블을 직접 만드실 필요가 없습니다! 스크립트가 모든 데이터를 훑어보고 예쁜 차트를 대신 그려줍니다.

1. `Tasks` 시트로 들어가서 **구글 스프레드시트 화면**을 열어둡니다.
2. 상단 메뉴바의 **[🚀 업무 시스템 관리]** 를 클릭합니다.
3. 드롭다운 메뉴에서 **`📊 대시보드 자동 생성 (새로고침)`** 버튼을 클릭합니다.
4. *(최초 실행 시 구글 권한 허용 창이 뜰 수 있습니다. 고급 -> 안전하지 않음으로 이동을 눌러 허용해 주세요)*
5. 3초 정도 기다리면 화면 하단에 파란색 탭 모양의 **`Dashboard`** 시트가 자동으로 만들어집니다.

### 💡 대시보드 갱신하기 (새로 고침)
- 누군가 업무 상태를 변경했거나, 새로운 업무가 등록된 경우
- 다시 한번 **[🚀 업무 시스템 관리] > [📊 대시보드 자동 생성]**을 누르시면, 알아서 기존 대시보드 시트를 지우고 **최신 현황이 반영된 새 차트**로 화면을 깔끔하게 복구(덮어쓰기) 해 줍니다.

---

## 📊 대시보드에서 볼 수 있는 통계 내용

1. **전체 업무 상태 분포 (원형 차트 - 좌측 상단)**
   - 팀 전체의 일이 현재 대기, 진행중, 완료 중에 몇 %씩 있는지 도넛 모양으로 직관적으로 파악할 수 있습니다. 

2. **담당자별 잔여 업무 로드 (막대 차트 - 우측 하단)**
   - 어떤 담당자에게 일이 가장 많이 몰려있는지(대기+진행중 상태인 개수), 한 눈에 보고 병목 현상을 해결할 수 있습니다.

---

## 🎯 번외: 모닝 브리핑 자동화 스위치 켜기

매일 아침 8시~9시 사이에 어제 남은 업무들을 슬랙으로 전송(브리핑) 받으시려면 아래 설정을 딱 한 번 켜주세요.

1. 상단의 확장 프로그램 > **Apps Script** 편집기로 들어갑니다.
2. 왼쪽 파일 목록에서 **`auto_automation.gs`** 파일을 더블 클릭해 엽니다.
3. 화면 상단의 **실행할 함수 선택** (디버그 버튼 옆의 드롭다운 영역)을 클릭합니다.
4. 목록에서 **`installMorningBriefingTrigger`** 를 찾아 선택합니다.
5. **[▶실행]** 버튼을 딱 한 번 누릅니다.

✅ 이제 슬랙에서 **[☀️ 주디의 아침 업무 브리핑]** 을 받아보실 수 있습니다.
