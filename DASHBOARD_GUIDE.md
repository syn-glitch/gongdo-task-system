# 공도 업무 관리 시스템 - 대시보드 구축 가이드

이 문서는 구글 스프레드시트의 자체 기능(피벗 테이블, 차트)을 활용하여 실시간 **업무 현황 대시보드**를 구축하는 방법을 안내합니다.

코드를 추가로 작성할 필요 없이 단 3분 만에 아름다운 현황판을 만들 수 있습니다.

---

## 📊 1. 전체 업무 상태 파이(원형) 차트 만들기
팀의 업무가 '대기', '진행중', '완료' 중 어디에 얼마나 분포해 있는지 한눈에 보는 차트입니다.

1. `Tasks` 시트로 이동합니다.
2. 상태 열(C열) 전체를 클릭하여 선택합니다.
3. 상단 메뉴에서 **삽입 > 차트** 를 클릭합니다.
4. 우측 `차트 편집기` 창에서:
   - **차트 유형**을 `원형 차트` (Pie chart)로 변경합니다.
   - **라벨**이 `상태` 로 잘 잡혀 있는지 확인합니다.
5. 차트가 완성되면 차트를 복사(Ctrl+C)하거나 이동시켜, 새로 만든 빈 시트(이름: `Dashboard`)에 예쁘게 배치합니다.

---

## 📈 2. 담당자별 진행 현황 막대 차트 (피벗 테이블 활용)
현재 누가 업무를 가장 많이 맡고 있는지, 완수는 얼마나 했는지 담당자별로 파악하는 표와 차트입니다.

1. `Tasks` 시트에서 전체 데이터 범위(A열 ~ I열 등)를 드래그하여 선택합니다.
2. 상단 메뉴에서 **삽입 > 피벗 테이블** (Pivot table)을 클릭합니다.
3. 생성 위치를 `새 시트`(또는 `Dashboard` 시트)로 선택하고 만들기 버튼을 누릅니다.
4. 우측 `피벗 테이블 편집기` 창에서 아래와 같이 설정합니다:
   - **행 (Rows):** 추가 버튼을 눌러 `담당자` 선택.
   - **열 (Columns):** 추가 버튼을 눌러 `상태` 선택. (대기, 진행중, 완료가 가로로 퍼집니다)
   - **값 (Values):** 추가 버튼을 눌러 `ID` (또는 업무 제목) 선택 후, 요약 기준을 **COUNTA (개수)** 로 설정.
5. (선택) 완성된 피벗 테이블 표를 전체 드래그한 뒤, **삽입 > 차트**를 다시 눌러 `누적 막대 차트` 형태로 띄우면 담당자별 업무 로드맵이 완벽하게 시각화됩니다.

---

## 🎯 3. 모닝 브리핑 자동화 설정 (선택 사항)
미리 작성해 둔 `generateMorningBriefing()` 코드를 매일 아침 알아서 실행하도록 스위치를 켜는 방법입니다.

1. 상단의 확장 프로그램 > **Apps Script** 편집기로 들어갑니다.
2. 왼쪽 파일 목록에서 **`auto_automation.gs`** 파일을 더블 클릭해 엽니다.
3. 화면 상단의 **실행할 함수 선택** (디버그 버튼 옆의 드롭다운 영역)을 클릭합니다.
4. 목록에서 **`installMorningBriefingTrigger`** 를 찾아 선택합니다.
5. **[▶실행]** 버튼을 딱 한 번 누릅니다. (권한 요구 시 승인)

✅ 설정이 끝났습니다! 이제 매일 아침 8시~9시 사이, 슬랙에 주디가 띄워주는 "오늘의 진행중/대기 업무" 아침 뉴스(모닝 브리핑)를 받아보실 수 있습니다.
