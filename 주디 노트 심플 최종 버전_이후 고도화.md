# 주디 노트 (Judy Note) 심플 최종 버전 및 향후 고도화 계획

## 1. 현재 상황 (심플 최종 버전)
**주디 노트**는 여러 차례 고도화를 거친 뒤, 가장 목적에 부합하는 **"심플하고 가벼운 메모장"** 형태로 최종 롤백 및 안정화되었습니다.

### 주요 핵심 기능
- **심플 텍스트 에어리어 (Simple Textarea)**: 무거운 TUI 마크다운 에디터를 제거하고 누구나 직관적으로 텍스트를 입력할 수 있는 기본 에디터 폼으로 롤백.
- **다중 포맷 아카이빙 지원 (Google Drive 통합)**:
  - 메모장 내에서 입력된 모든 텍스트는 드라이브 내 `년-월_업무일지.md` 텍스트형 파일에 자동 추가(Append) 됩니다.
  - 외부에서 구글 드라이브 앱을 통해 직접 생성한 **구글 문서(Google Docs)** 형식 또한 정상적으로 파싱하여 과거 기록 뷰어에 호환 출력됩니다.
- **슬랙(Slack) 매직 링크 인증**: 비밀번호 로그인, 구글 Oauth 권한 팝업 없이 슬랙 봇을 통해 1회용 토큰(Magic Link)을 검증하여 완벽한 자동 로그인 시스템을 구축했습니다.
- **AI 융합 모듈 (Claude AI)**:
  - ✨ **AI 업무 추출**: 적은 메모 내용 중 일해야 할 Task를 AI가 선별하여 공도 업무 시스템 구글 시트로 자동 전송하고 슬랙 알림을 보냅니다.
  - ✨ **AI 내용 요약**: 과거의 긴 문서를 읽을 때 요약 팝업을 통해 핵심만 빠르게 훑어볼 수 있게 지원합니다.
  - 🤖 **글로벌 RAG 검색 & 라우팅**: 원하는 키워드 입력 만으로 모든 과거 문서를 즉시 검색하여 하이라이트를 제공합니다.

## 2. 향후 프로젝트 고도화 계획 (Future Enhancements)
현재 안정화된 버전을 해치지 않는 선에서, 아래 사항들을 미래 과제로 선정합니다.

### 2.1 에디터 내용 수정 모드 (Edit Mode) & 덮어쓰기 기능
- **설명**: 현재 텍스트 뷰어(Read Mode)로만 볼 수 있는 과거의 기록을 에디터에서 직접 타건하여 원본 파일을 덮어쓰기(Update) 하는 기능입니다.
- **해결 과제**:
  - 구글 드라이브(MD) 혹은 구글 문서(Doc) 원본 문서를 훼손시키지 않으면서 특정 시간대(\`[08:00 AM]\`) 블록만을 안전하게 찾아 치환하는 정규표현식 및 파싱 로직의 완성.
  - 구글 문서 (Google Workspace) 권한 제어 이슈 해결.

### 2.2 AI 챗봇과의 양방향 인터페이스 (AI Chat & Ask)
- **설명**: 단순히 메모 전체를 요약하는 팝업 기능에서 그치지 않고, 본문 내용 중 특정 구역을 드래그하여 물어보거나 메모장 사이드에 AI와 자유롭게 대화할 수 있는(티키타카) 채팅 영역을 구축합니다.
- **비전**: 단순 메모장에서 **'지능형 워크스페이스(Intelligent Workspace)'** 로 한 단계 도약하게 됩니다. 노션(Notion)의 AI 블록과 유사한 사용자 경험을 웹에 구현하는 것이 핵심입니다.

### 2.3 팀 관리자 대시보드 인테그레이션 (Dashboard Data Tracking)
- **설명**: 현재 수집되고 있는 AI 업무 추출 데이터, 개개인의 `Tasks` 시트 데이터를 바탕으로 구글 시트 기반 시각화 대시보드를 연동시킵니다.
- **비전**: 각 팀원들의 메모 및 업무 일지가 부서장 및 리드에게 매일 아침 자동 요약(Morning Briefing)되어 조직의 생산성을 측정하는 주요 지표로 활용되게 합니다.
