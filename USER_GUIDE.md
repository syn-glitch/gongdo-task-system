# 🐰 주디 (Judy) AI 에이전트 워크스페이스 — 사용자 가이드

> **최종 업데이트**: 2026-02-24 | **버전**: v2.0 (Workspace 통합)

---

## 📌 주디란?

**주디(Judy)**는 공도 팀의 AI 업무 관리 에이전트입니다. 슬랙과 웹 워크스페이스를 통해 업무 등록, 메모 기록, AI 분석, 일간/주간 리포트를 자동화합니다.

### 접속 방법

| 방법 | 설명 |
|---|---|
| **슬랙 DM** | Judy Ops 봇에게 `노트` 또는 `메모장`이라고 DM 전송 → 매직 링크 수신 |
| **슬랙 명령어** | `/주디 내업무` 입력 → 대시보드 바로가기 버튼 수신 |

> 🔒 **보안**: 매직 링크는 **10분 후 만료**, **1회 클릭 시 소멸**됩니다. 구글 로그인 없이 슬랙 인증으로만 접속합니다.

---

## 🏠 워크스페이스 구성

워크스페이스는 **2개 탭**으로 구성되어 있습니다:

### 📝 내 노트 탭
메모, 회의록, 아이디어를 기록하고 AI가 요약·분석합니다.

| 영역 | 기능 |
|---|---|
| **좌측 사이드바** | 날짜별 폴더 구조로 과거 메모 조회 |
| **작성 영역** | 새 메모 작성 (Cmd+Enter로 빠른 저장) |
| **검색** | 키워드로 전체 메모 검색 (사이드바 상단) |

### 📊 내 업무 탭
내 업무를 한 눈에 관리하는 대시보드입니다.

| 영역 | 기능 |
|---|---|
| **통계 카드** | 전체/진행중/대기/완료 업무 수 실시간 표시 |
| **업무 테이블** | 프로젝트, 제목, 상태, 마감일 등 한눈에 조회 |
| **새 업무 등록** | ➕ 버튼으로 직접 등록 가능 |

## ✨ 핵심 기능 사용법

### 1. 메모 작성 & 저장

1. `📝 내 노트` 탭에서 메모를 작성합니다
2. **[일반 저장]** 버튼 또는 `Cmd + Enter` / `Ctrl + Enter`로 저장
3. 저장된 메모는 구글 드라이브에 자동 백업됩니다

---

### 2. 🐰 드래그 → 업무 등록 (AI 추출)

메모에서 업무가 될 만한 내용을 발견했을 때:

1. 해당 텍스트를 **마우스로 드래그(선택)** 합니다
2. 선택 영역 위에 **🐰 업무 등록** 보라색 버튼이 나타납니다
3. 버튼을 클릭하면 AI가 내용을 분석합니다
4. 자동으로 **📊 내 업무** 탭으로 이동하며, 업무 등록 모달이 열립니다
5. AI가 추출한 **제목, 내용, 마감일**이 미리 채워져 있으므로 확인 후 **[등록하기]** 클릭!

> 💡 읽기 모드(과거 메모 조회)에서도 동일하게 작동합니다.
> ⚠️ 5글자 이상 선택해야 버튼이 나타납니다.

---

### 3. ✨ AI 내용 요약

과거 메모를 읽을 때 긴 내용을 빠르게 파악하고 싶다면:

1. 좌측 사이드바에서 과거 메모를 클릭합니다
2. 상단의 **[✨ AI 내용 요약]** 버튼을 클릭합니다
3. AI가 핵심을 1~3줄로 요약하여 보여줍니다

---

### 4. 업무 등록 & 관리

#### 직접 등록
1. `📊 내 업무` 탭에서 **[➕ 새 업무 등록]** 버튼 클릭
2. 프로젝트, 업무 제목, 내용, 마감일, 상태를 입력
3. **[등록하기]** 클릭 → 구글 시트에 자동 저장 & 슬랙 알림 발송

#### 상태 변경
- 업무 행을 클릭하면 상세 모달이 열립니다
- 상태(대기/진행중/완료) 변경 → **[저장]** 클릭

---

### 5. 슬랙 DM 기능

Judy Ops 봇에게 DM으로 다양한 기능을 사용할 수 있습니다:

| 입력 | 기능 |
|---|---|
| `노트` / `메모장` / `링크` | 🔗 매직 링크 발급 (웹 워크스페이스 접속) |
| `오늘 보여줘` / `오늘 기록 검색` | 📋 오늘 작성한 메모 조회 |
| 일반 질문 | 🤖 AI가 업무 데이터 기반으로 답변 |

## ⚠️ 자주 묻는 질문 (FAQ)

### Q. 매직 링크가 만료되었다고 뜹니다
> 링크는 10분 후 또는 1회 클릭 시 소멸됩니다. 슬랙에서 `노트`를 다시 입력하여 새 링크를 받으세요.

### Q. 업무가 대시보드에 안 보입니다
> 상단 탭에서 **[진행중]** 또는 **[완료]** 탭을 확인하세요. 새로고침 버튼(🔄)을 눌러보세요.

### Q. 모바일에서도 사용 가능한가요?
> 네! 반응형 디자인으로 모바일에서도 최적화되어 있습니다. 좌측 상단 ☰ 버튼으로 사이드바를 여닫을 수 있습니다.

### Q. AI가 추출한 업무 내용이 정확하지 않습니다
> AI 추출 결과는 참고용입니다. 등록 전 모달에서 직접 수정한 후 등록하세요.
