## 기록 규칙

1. 업무 완료 보고(completion_report) 시 반드시 이 파일에 한 행을 추가한다
2. 토큰 수량은 근사치(약 XX,XXX)로 기록한다
3. 참여 팀원은 실제 작업에 참여한 에이전트만 기록한다
4. 월별 요약은 해당 월의 마지막 기록 시 업데이트한다
5. 이전 기록은 절대 삭제하지 않는다
