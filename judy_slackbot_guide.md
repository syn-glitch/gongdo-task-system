# 주디 슬랙봇 (Judy Slackbot) 사용 가이드

**주디 슬랙봇**은 공도 팀의 슬랙 워크스페이스 내에서 직접 업무를 등록하고, 개인적인 메모를 구글 드라이브에 빠르게 저장하며, 오늘의 기록을 다시 불러올 수 있는 만능 인공지능 개인 비서입니다.

## 🌟 핵심 기능
1. **`/주디` 슬래시 커맨드**: 슬랙 어디서든 `/주디` 를 입력하면 상세한 속성을 갖춘 모달(팝업) 창이 뜹니다.
2. **개인 DM 빠른 메모 저장**: 주디 봇과의 1:1 개인 DM(Direct Message) 창에서 채팅을 치면, 그 내용이 즉시 본인의 '주디 노트(구글 드라이브)'에 시간과 함께 찍혀 영구 보관됩니다. (AI가 답장하지 않고 조용히 기록만 합니다.)
3. **오늘의 메모 검색**: DM 창에 "오늘 메모 보여줘" 라고 치면, 오늘 하루 동안 주디봇에게 던져두었던 모든 메모들을 시간순으로 예쁘게 모아서 보여줍니다.
4. **공개 채널 AI 응답**: 개인 DM이 아닌 팀원들이 모여있는 일반 채널에서 `@주디` 를 태그하고 질문하면, 우리 팀의 업무 맥락(구글 시트)을 이해하고 있는 클로드(Claude) AI가 스마트한 답변을 제공합니다.
5. **업무 배정 및 알림 DM**: 누군가 나를 담당자로 지정하여 업무를 생성하면, 주디봇이 내게 1:1 DM으로 "이러이러한 업무가 배정되었습니다!" 라고 친절하게 알려줍니다.

## 📝 사용 방법

### 1. 정식 업무 등록하기 (모달창)
- 채팅 입력칸에 `/주디` 라고 타이핑 후 전송합니다.
- 뜨는 모달창에 프로젝트명, 업무 제목, 상세 설명을 기입합니다.
- 필요시 마감일(달력)과 당담자(슬랙 팀원 목록)를 선택하고 `[등록 완료하기]` 를 누르세요.

### 2. 개인 아이디어 던져놓기 (DM 메모장 저장기능)
- 슬랙 좌측 하단 '앱' 이나 '다이렉트 메시지' 에서 **Judy Apps** 봇을 찾아 클릭합니다.
- 생각나는 문장이나 링크, 복사해둔 텍스트를 대충 전송합니다.
- 카카오톡 '나에게 보내기' 처럼 활용하세요! 봇이 방해 답변을 달지 않고 조용히 드라이브에 쌓아줍니다.

### 3. 오늘 쓴 메모 다시보기
- 주디 봇 DM 창에서 아래와 같이 말해보세요! (키워드 매칭 방식입니다)
  > "오늘 작성 메모 보여줘"
  > "오늘 기록 알려줘"
  > "오늘 뭐 적었더라"
- 그 즉시 `📅 2026-02-22 업무 기록 내역입니다:` 라면서 불릿 리스트로 예쁘게 모아줍니다.

## 💡 팁 (Tip)
슬랙 채널에서 팀원들과 긴 채팅으로 논의를 마치고 이것이 '업무'로 결정 났을 때, 번거롭게 복사/붙여넣기 할 필요 없이 해당 슬랙 메시지 우측의 `[:점 3개 메뉴:] -> [앱 바로가기] -> [이 메시지로 새 업무 등록]` 버튼을 누르면 메시지 원문이 그대로 모달창에 복사되어 들어갑니다!
