# 👩‍💻 첫 AI 팀원 주디 (Judy) 활용 가이드

## ✨ 주디(Judy) AI 팀원 소개
안녕하세요! 우리 팀의 첫 AI 에이전트 팀원, **주디(Judy)**입니다.
주디는 팀원들이 업무에만 온전히 집중할 수 있도록, 번거로운 **업무 등록, 메모 정리, 일정 관리**를 도와주는 똑똑한 비서입니다. 슬랙(Slack) 안에서 모든 것을 처리할 수 있으며, 흩어진 대화를 모아주고, 대충 적은 메모에서도 핵심 업무를 척척 뽑아내어 구글 시트와 드라이브에 안전하게 보관해 줍니다. 

주디와 함께 더 효율적이고 가벼운 업무 환경을 경험해 보세요! 🚀

---

## 🛠 주요 기능 및 사용 방법

### 1. 📊 내 업무 대시보드 (강력 추천!)
내가 해야 할 일이나 구글 시트에 등록된 밀린 업무들을 슬랙 환경의 제약 없이 빠르고 시원하게 **웹 대시보드**에서 조회하고 관리합니다. (로그인 필요 없음!)
* **사용 방법**:
  1. 슬랙 아무 채팅창에 `/주디 내업무` (또는 `/주디 내 업무`)를 입력합니다.
  2. 주디가 보내주는 **[📊 내 업무 대시보드 열기] 버튼**을 누릅니다.
  3. 로그인 스킵! 내 아이디로 자동 인증되어, 오직 내게 할당된 일감만 나타납니다.
  4. 웹 화면 상단에서 **[+ 새 업무 등록]** 버튼을 눌러 프로젝트명, 마감일, 상태를 빠르게 추가합니다.
  5. 리스트 항목들의 **상태(대기/진행/보류) 드롭다운을 즉각적으로 수정**할 수 있으며, 수정된 상태는 구글 시트에 실시간 저장됩니다.
  6. 제목 옆 🚨, 🔥, ⚠️ 배지를 통해 마감기한(이미 지남/오늘/내일)을 직관적으로 확인하세요.

### 2. 📝 슬랙 모달 업무 등록 
슬랙에서 웹으로 나가지 않더라도 즉각적으로 간단한 태스크를 부여받은 경우 슬랙 내에서 다이렉트로 등록합니다.
* **사용 방법**: 
  1. 슬랙 메시지 입력창에 `/주디` 를 입력하고 전송합니다.
  2. 팝업되는 **'업무 등록 폼(Modal)'**에서 내용(프로젝트, 담당자, 시작일/종료일 등)을 입력하고 **[제출]**을 누릅니다.
  3. 주디가 알아서 공용 구글 시트에 새 행으로 깔끔하게 등록해 주고, 구조적 식별 ID(예: GONG-001)도 부여해줍니다. 구글 캘린더에도 등록한 마감날짜에 맞게 스케줄을 자동 기입합니다.
  4. 본인이 남에게 업무를 할당(`담당자 지정`)했다면, 주디가 해당 담당자에게 DM 봇 메시지로 즉시 알림을 쏴줍니다!

### 3. 💬 스레드/메시지 가져오기 (메시지 액션)
슬랙으로 대화하다가 "이건 업무로 등록해야 해!" 또는 "이 내용은 메모해 둬야겠다!" 싶을 때, 대화를 그대로 끌어옵니다. 복붙이 필요 없습니다!
* **사용 방법**:
  1. 저장하고 싶은 슬랙 메시지나 스레드에 마우스를 올리고, 우측의 **[점 3개(더보기) 버튼]**을 클릭합니다.
  2. 메뉴에서 **[주디 - 업무로 가져오기]** 혹은 **[주디 - 메모로 가져오기]**를 선택합니다.
  3. 메시지 내용이 '상세 내용' 텍스트 칸에 자동 입력된 상태로 모달이 열립니다. 마감일과 담당자만 찍어주고 제출 버튼을 눌러 즉시 DB화 합니다.

### 4. 🔗 팀원별 특권! 주디 노트(웹앱) 매직 링크
팀원마다 독립된 자기만의 드라이브 아카이브 폴더(YYYY-MM)를 가지며, 넓은 뷰에서 메모를 남길 수 있는 곳입니다. 귀찮은 ID/비밀번호나 구글 로그인이 필요 없는 **통합 매직 링크(Magic Link)** 방식을 사용합니다.
* **사용 방법**:
  1. 주디와의 1:1 채팅방(DM)에서 아무 말이나 치거나 "노트"라고 입력합니다. 
  2. 주디가 내 전용 보안 토큰이 포함된 **"나만의 웹앱 링크"**를 보내주면 누르기만 하세요! (1초 컷 로그인)

### 5. ✍️ 주디 노트 (Web Notepad) 작성 및 원클릭 AI 추출
회의록이나 아이디어를 타자 치기 편한 넓은 화면에서 작성하고, 내 드라이브에 마크다운 문서 파일로 영구 저장합니다.
* **[일반 저장] (단축키: Cmd/Ctrl + Enter)**: 작성한 텍스트를 즉시 내 폴더에 안전하게 백업합니다.
* **[✨ AI 업무 추출] (단축키 없음 - 버튼 클릭)**: 텍스트를 적어두고 별 모양 AI 버튼을 누르면, 주디봇이 전체 글을 읽고 **"당신이 해야 할 행동/업무 요건"**만 추출해서 자동으로 시트(DB)에 '대기 업무'로 목록업 해줍니다. 

### 6. 📂 과거 주디 노트 열람 및 요약 (사이드바 리딩)
과거에 내가 썼던 노트들을 편하게 열람하고, 긴 글의 맥락을 AI 요약본으로 받아봅니다.
* **사용 방법**:
  1. 주디 노트 **좌측 사이드바**(모바일은 상단 ☰ 메뉴) 폴더 트리에서 과거 기록 일자를 클릭합니다.
  2. 뷰어 우측 상단의 **[✨ AI 내용 요약]** 버튼을 누르면, 주디가 5~10초 만에 해당 글의 핵심만 3~5줄로 요약해서 팝업으로 띄워줍니다.
