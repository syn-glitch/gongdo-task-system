# 주디 노트 (Judy Note) 사용 가이드

**공도 업무 관리 주디 노트**는 개인적인 아이디어, 회의록, 투두 리스트 등을 구글 드라이브에 안전하게 보관하고, 필요시 AI의 힘을 빌려 요약 및 업무 추출을 할 수 있는 웹 기반 스마트 메모장입니다.

## 🌟 주요 기능
1. **마크다운 기반 일자별 자동 분류**: 메모를 저장하면 자동으로 구글 드라이브 지정된 폴더에 `[월별 폴더] > [일자별 파일]` 형태로 예쁘게 누적 저장됩니다.
2. **다크모드 / 라이트모드 지원**: 우측 상단 ☀️/🌙 버튼을 클릭하여 시력 보호를 위한 테마 전환이 가능합니다. (마지막 설정 기억)
3. **AI 내용 요약 (Pop-up)**: 과거에 적은 긴 메모를 다시 읽어볼 때, 우측 상단 `✨ AI 내용 요약` 버튼을 누르면 클로드(Claude)가 핵심만 3~4줄로 팝업 요약해 줍니다.
4. **AI 업무 추출**: 단순 메모 저장을 넘어, 메모 내용 중 '실제 수행해야 할 업무(Task)'가 있다면 `✨ AI 업무 추출` 버튼을 클릭하세요. AI가 자동으로 업무를 분석하여 시스템(구글 시트)에 기입해 줍니다.
5. **강력한 전역 검색**: 좌측 사이드바 상단 검색창에 키워드를 입력하고 `Enter`를 누르면, 과거의 모든 메모 파일 내용을 싹 뒤져서(RAG 방식) 해당 단어가 포함된 문맥을 하이라이트 해서 보여줍니다.

## 📝 사용 방법
1. 브라우저에서 **주디 노트 웹 앱 url** 로 접속합니다.
2. 가장 먼저 우측 상단 **[작성자 선택]** 드롭다운에서 본인의 이름을 선택합니다. (최초 1회 선택 시 브라우저에 기억됨)
3. 좌측 사이드바에 과거에 본인이 썼던 폴더 트리가 불러와지는 것을 확인합니다.
4. **새 메모 작성**: 좌측 상단 `+ 새 메모` 를 누르고, 중앙 텍스트창에 내용을 자유롭게 적습니다.
   - 단축키: `Cmd + Enter` (또는 `Ctrl + Enter`) 로 빠르게 [일반 저장] 할 수 있습니다.
5. **과거 메모 열람**: 좌측 폴더 트리에서 특정 날짜를 클릭하면 해당 날짜에 적었던 모든 메모 내역이 우측 화면에 펼쳐집니다.

## ⚠️ 주의 사항
- `일반 저장`은 구글 드라이브 마크다운 파일에만 추가되며 팀원들에게 슬랙 알림이 가지 않습니다. 혼자 보기 위한 기록장 용도입니다.
- AI 기능(`요약` 및 `업무 추출`)은 처리 시간이 3~5초 정도 소요될 수 있습니다. 버튼이 다시 활성화될 때까지 기다려 주세요.
