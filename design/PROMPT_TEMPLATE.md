# 비전공자를 위한 AI 협업(프롬프트) 가이드: 슬랙-구글시트 업무 자동화 시스템

이 문서는 비개발자(비전공자)가 ChatGPT나 Claude, 혹은 AI 코딩 비서(Gemini 등)를 활용하여 **"지금 완성된 공도 업무 관리 시스템"을 밑바닥부터 똑같이 다시 만들고자 할 때** 복사해서 붙여넣기 좋게 정리해 둔 **프롬프트(명령어) 템플릿**입니다.

---

## 💡 AI와 코딩할 때 핵심 꿀팁
1. **한 번에 다 해달라고 하지 마세요!** (가장 중요)
   - "구글 시트랑 슬랙이랑 캘린더 연동하고 AI까지 붙여서 업무 시스템 만들어줘"라고 하면 AI가 방대한 코드를 쏟아내다 중간에 멈추거나 오류가 납니다.
   - 반드시 아래처럼 **1단계 완수 후 작동 확인 ➡️ 그다음 2단계 진행** 순서로 쪼개서 요청하세요.
2. **에러가 나면 화면을 그대로 복사해서 주세요.**
   - 에러로 빨간 글씨가 나오면 혼자 끙끙대지 말고 `"이런 에러가 나옵니다: (에러 메시지 복붙) 해결해 줘"`라고 하면 AI가 원인을 찾아 코드를 수정해 줍니다.
3. **제가 모르는 전문 용어는 쉽게 풀어서 설명해달라고 하세요.**

---

## 🚀 단계별 프롬프트 (복사해서 AI에게 차례대로 입력하세요)

### [1단계] 구글 스프레드시트 기초 공사
> **프롬프트 복사하기:**
```text
안녕! 나는 코딩을 잘 모르는 기획자야. 우리 팀의 업무를 관리할 수 있는 데이터베이스를 구글 스프레드시트에서 만들고 싶어.
구글 앱스 스크립트(GAS)를 써서 내가 메뉴 버튼을 누르면 3개의 시트('Tasks', 'Projects', 'Users')가 자동으로 생성되게 코드를 짜줄래?

- Tasks 시트에는 '업무 ID, 업무 유형, 상태, 프로젝트, 업무 제목, 상세 내용, 담당자, 마감일, 최근 수정일' 헤더가 있어야 해.
- 상태 칸에는 '대기, 진행중, 완료, 보류' 드롭다운(선택 메뉴)이 들어가게 해 줘.
- 내가 업무 상태를 바꾸거나 내용을 수정하면 '최근 수정일' 칸에 현재 시간이 자동으로 찍히고, '프로젝트'를 입력하면 '업무 ID'(예: MKT-001)가 자동으로 새로 생기는 온에디트(onEdit) 기능도 같이 짜줘.
```

### [2단계] 슬랙(Slack) 일방향 알림 연동
> **프롬프트 복사하기:**
```text
(1단계 코드가 잘 작동해!) 
이제 누군가 구글 시트의 'Tasks' 시트에서 업무의 상태를 '대기'나 '진행중'으로 바꿨을 때, 우리 팀 슬랙 채널로 알림이 오게 만들고 싶어. 

- 'Projects' 시트에는 프로젝트명과 알림을 보낼 슬랙 채널 ID가 매핑되어 있어. 지정된 프로젝트 채널로 알림이 가게 해줘.
- 슬랙 메시지는 "새로운 업무가 등록(혹은 변경)되었습니다"라는 말과 함께 담당자, 업무 제목, 마감일을 예쁘게 묶어서 보내줘.
- 슬랙 Incoming Webhook이나 Bot 토큰(chat.postMessage)을 사용하게 해 주고, 이 스크립트를 어떻게 세팅해야 하는지(슬랙 개발자 콘솔 설정법) 비전공자도 알기 쉽게 튜토리얼을 상세히 적어줘.
```

### [3단계] 슬랙에서 바로 업무 등록하기 (모달창 생성)
> **프롬프트 복사하기:**
```text
(알림이 잘 와!)
이번에는 구글 시트를 열지 않고도 슬랙에서 바로 업무를 등록하고 싶어.

- 누군가 슬랙 채팅창에 `/주디` (또는 /업무) 라고 슬래시 커맨드를 치면, 슬랙 팝업창(모달)이 뜨게 해 줘.
- 모달 창에는 '프로젝트명', '업무 제목', '상세 내용', 그리고 달력에서 고를 수 있는 '마감일(DatePicker)' 블록이 들어가면 좋겠어.
- 모달에서 '등록'을 누르면 구글 시트 'Tasks' 시트에 새 줄이 추가되게 해고, 에러(타임아웃 3초 에러)가 나지 않도록 PropertiesService나 캐시를 써서 백그라운드(비동기)로 시트에 저장하는 로직으로 안전하게 코드를 짜 줘.
- 이 코드를 '웹 앱'으로 배포해서 슬랙에 연결하는 방법도 차근차근 알려줘.
```

### [4단계] 구글 캘린더 동기화 연동
> **프롬프트 복사하기:**
```text
(슬랙에서 등록이 잘 돼!)
이제 마감일 관리를 위해 구글 캘린더와 연동하려고 해.

- 'Tasks' 시트에 마감일이 입력된 업무는 해당 담당자의 구글 캘린더에 당일 '종일 일정(All Day Event)'으로 알아서 등록되게 스크립트를 짜줘.
- 만약 시트에서 마감일 날짜를 수정하면 캘린더 일정 날짜도 같이 바뀌고, 업무 상태가 '완료'나 '보류'로 바뀌거나 마감일을 삭제하면 캘린더 일정에서도 알아서 삭제되게 해 줘. 
- 이전에 만들어둔 3단계 모달창에서 등록할 때도 캘린더에 바로 들어가도록 기존 코드와 연결해줘.
```

### [5단계] 인공지능(Claude API) 기능 붙이기: 리포트 & 챗봇
> **프롬프트 복사하기:**
```text
(캘린더도 완벽해!)
이제 내 구글 시트에 인공지능(Claude 3 API)를 연결해서 스마트한 비서를 만들고 싶어. 

1. **AI 챗봇 기능:** 슬랙에서 봇 주디에게 개인 메시지(DM)나 `@주디` 멘션으로 "오늘의 주요 업무 뭐냐"고 자연어로 물어보면, 구글 시트 데이터를 읽어서 클로드 AI가 상황에 맞게 10초 안에 대답해 주도록 해줘. 슬랙 Event Subscriptions(이벤트 구독) 설정법도 같이 알려줘.
2. **모닝 브리핑 기능:** 매일 아침 '진행중'이거나 '대기' 중인 업무만 추려서 클로드 AI가 활기찬 아침 인사와 함께 요약해 주는 `generateMorningBriefing` 함수도 만들어 줘.
- 슬랙의 3초 타임아웃을 방어하기 위해 CacheService 등을 써서 안전하게 데이터를 처리해 줘.
```

### [6단계] 1-Click 자동 대시보드 만들기
> **프롬프트 복사하기:**
```text
(마지막 단계야!)
구글 시트에 수많은 업무 데이터가 쌓였는데, 이걸 차트로 한눈에 파악하고 싶어. 

내가 직접 피벗 테이블을 만지지 않고도 상단의 커스텀 메뉴바에서 '대시보드 자동 생성'을 클릭하면, 'Tasks' 시트 데이터를 바탕으로 'Dashboard'라는 예쁜 새 시트가 만들어지는 코드를 작성해 줘.
- 차트 1: 전체 업무의 상태(대기/진행/완료 등) 비율을 보여주는 원형 파이 차트
- 차트 2: 각 담당자별로 쌓여있는 '대기+진행중' 업무 할당량을 보여주는 막대 차트
SpreadsheetApp.newChart() 빌더를 이용해 코딩해 줘!
```

## 📌 마무리 조언
이 시스템은 구글 솔루션(구글 시트, 캘린더, 앱스 스크립트)과 슬랙 앱 시스템이 복잡하게 얽혀 있습니다. 
따라서 AI가 짜주는 **코드**뿐만 아니라, AI가 알려주는 **[관리자 웹사이트(슬랙 API 콘솔 등)에서 직접 마우스로 클릭해서 설정해야 하는 세팅법]**을 꼼꼼히 따라 하는 것이 오류를 막는 지름길입니다. 화이팅입니다!
