4. **원인 추정**: 초기 설계 시 1:1 업무 할당 모델만 고려, 다중 관계 미구현

## 영향 범위
이 기능 추가가 영향을 미치는 범위:

1. **데이터 레이어**
   - 스프레드시트 스키마 변경 필요

   - 마이그레이션: 기존 단일 담당자 데이터 보존

2. **UI/UX 레이어**
   - 업무 카드 모달 HTML 수정
   - 담당자 추가/삭제 버튼 구현
   - 멀티셀렉트 또는 태그 입력 컴포넌트

3. **비즈니스 로직**

4. **연동 시스템**
