```javascript
function rateLimitCheck(userId) {
  var cache = CacheService.getScriptCache();
  var key = 'rate_' + userId;
  var count = cache.get(key) || 0;

  if (count > 10) {
    throw new Error('Rate limit exceeded');
  }

  cache.put(key, parseInt(count) + 1, 60); // 60초 TTL
}
```

**계산**:
- 월 10건 해결 목표
- 현재 백로그 7건 = 41시간
- 월 10건 = **약 59시간** (41/7 * 10)

**문제**:
- 4명 팀 × 월 160시간 (주 40시간 × 4주) = **640시간**
- 하지만 기술 부채만 처리하는 것이 아님:
  - QA 팀 요청 대응: 월 5건 × 4시간 = 20시간
  - 자비스 팀 협업: 월 3건 × 6시간 = 18시간
  - 팀 미팅 및 문서화: 월 20시간
  - **실제 가용 시간**: 640 - 20 - 18 - 20 = **582시간**

**결론**: 월 10건은 가능하나, **여유 시간이 거의 없음** (582 - 59 = 523시간 남음)

**제안**: 목표를 "월 8건"으로 하향 조정 또는 인력 증원
