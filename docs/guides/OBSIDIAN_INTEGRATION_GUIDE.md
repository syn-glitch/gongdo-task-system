# 💎 [Guide] 주디 워크스페이스 - 옵시디언(Obsidian) 초간편 연동 가이드

**작성자**: 벙커팀장 박기획
**작성일**: 2026-03-01
**대상**: 옵시디언을 주 업무 도구로 사용하는 대표님 및 팀원

---

## 🚀 0. 연동 원리
주디 워크스페이스는 메모를 작성할 때 **구글 드라이브**의 특정 폴더에 `.md` (마크다운) 파일을 자동으로 생성/업데이트합니다. 옵시디언은 로컬의 `.md` 파일을 읽는 도구이므로, 구글 드라이브와 로컬 저장소를 동기화하기만 하면 즉시 연동됩니다.

## 🛠️ 1. 사전 준비 (최초 1회)
1.  **Google Drive 데스크톱 앱 설치**: [구글 드라이브 다운로드](https://www.google.com/intl/ko_kr/drive/download/) 페이지에서 Mac용 앱을 설치합니다.
2.  **내 드라이브 동기화**: 설치 후 로그인을 완료하면 내 컴퓨터에 `Google Drive`라는 가상 드라이브(또는 폴더)가 생깁니다.

## 🔗 2. 옵시디언(Obsidian) 연결하기
1.  **동기화 폴더 위치 확인**: 
    - 주디가 메모를 저장하는 구글 드라이브 폴더의 경로를 로컬 파일 탐색기(Finder)에서 찾습니다.
    - 예: `/Users/사용자명/Library/CloudStorage/GoogleDrive-이메일/내 드라이브/주디노트_아카이브/`
2.  **옵시디언 볼트(Vault)로 열기**:
    - 옵시디언 앱을 실행합니다.
    - `Open folder as vault` (폴더를 볼트로 열기)를 선택합니다.
    - 위 1번에서 확인한 `주디노트_아카이브` 폴더를 선택합니다.

## ✨ 3. 연동 후 활용 팁
- **실시간 동기화**: 주디 워크스페이스(웹)에서 메모를 저장하면, 수초 내로 옵시디언 파일 목록에 새로운 내용이 나타납니다.
- **그래프 뷰 활용**: 주디에 작성한 월별/날짜별 업무 기록들이 서로 연결되는 지식 그래프를 옵시디언에서 바로 확인할 수 있습니다.
- **백업 걱정 제로**: 모든 데이터는 구글 드라이브(클라우드)와 내 컴퓨터(로컬) 두 곳에 동시에 안전하게 보관됩니다.
