# 📈 토큰 대시보드 — 사용 가이드

---

## 1. 접속 방법

1. **Judy Workspace**에 로그인한다 (슬랙 매직링크 또는 세션)
2. 상단 GNB에서 **"📈 토큰"** 탭을 클릭한다
3. 모바일에서는 하단 네비게이션의 **"📈"** 아이콘을 탭한다

> **참고**: 토큰 탭은 허용된 사용자(송용남, 정혜림)에게만 표시된다. 다른 사용자에게는 탭 자체가 보이지 않는다.

---

## 2. 대시보드 화면 구성

### 2.1 기간 필터

화면 상단에 **1일 / 7일 / 15일 / 30일** 버튼이 있다.

| 버튼 | 조회 범위 | 용도 |
|------|----------|------|
| **1일** | 오늘 하루 | 실시간 사용량 확인 |
| **7일** | 최근 1주일 | 주간 트렌드 파악 |
| **15일** | 최근 15일 | 반월 추이 분석 |
| **30일** | 최근 30일 | 월간 비용 산출 |

기본값은 **1일**이다. 버튼을 클릭하면 즉시 해당 기간의 데이터로 갱신된다.

### 2.2 요약 카드 (4개)

| 카드 | 설명 | 예시 |
|------|------|------|
| **총 토큰** | 선택 기간 내 소모된 입력+출력 토큰 합계 | 11,779 |
| **API 호출** | 선택 기간 내 Claude API 호출 횟수 | 1회 |
| **예상 비용** | 토큰 단가 기반 USD 환산 비용 | $0.042 |
| **일 평균** | 기간 내 하루 평균 토큰 소모량 | 11,779 |

**비용 산출 기준** (기본값):
- Input: $3 / 1M tokens (Sonnet)
- Output: $15 / 1M tokens (Sonnet)

### 2.3 일별 사용량 차트

- **파란색 막대**: Input 토큰 (요청 시 소모)
- **주황색 막대**: Output 토큰 (응답 시 소모)
- 날짜별로 사용량 추이를 시각적으로 비교할 수 있다
- 차트에 마우스를 올리면 상세 수치가 표시된다

### 2.4 기능별 사용량 (파이 차트)

어떤 AI 기능이 토큰을 가장 많이 소모하는지 비율로 표시한다.

| 기능명 (한글) | 실제 함수명 | 설명 |
|-------------|-----------|------|
| 채팅 | processJudyWebChat | 주디 웹 채팅 |
| 브리핑 | getDailyBriefingForWeb | 웹 브리핑 조회 |
| 업무 추출 | extractTasksWithClaude | AI 업무 파싱 |
| 메모 분석 | parseTaskFromMemoWeb | 메모→업무 변환 |
| 요약 | summarizeMemoContent | 메모 내용 요약 |
| 청크 요약 | summarizeLongText_chunk | 긴 텍스트 분할 요약 |
| 통합 요약 | summarizeLongText_merge | 분할 결과 통합 |
| 일일 보고 | askClaude | 슬랙 일일 리포트 |

### 2.5 팀원별 사용량 테이블

사용자별로 총 토큰, 호출 횟수, 비중(%)을 표시한다.

### 2.6 최근 API 호출 로그

최신 50건의 API 호출 기록을 표시한다:
- 시간 / 기능 / 사용자 / Input / Output / 합계

## 4. 데이터 해석 가이드

### Input 토큰이 높은 경우
- 시스템 프롬프트 + 컨텍스트(업무/메모/일정 데이터)가 포함되므로 자연스럽게 높다
- 채팅 기능은 사용자의 전체 데이터를 컨텍스트로 전달하므로 Input이 특히 크다

### Output 토큰이 높은 경우
- AI의 응답이 길었음을 의미한다
- 브리핑, 요약 기능에서 상대적으로 높을 수 있다

### 비용 절감 팁
- 불필요한 반복 질문 줄이기
- 캐시가 작동하므로 동일 질문은 5분 이내 재질문 시 API 호출 없음

## 5. FAQ

**Q: 토큰 탭이 보이지 않아요.**
A: 허용된 사용자(송용남, 정혜림)에게만 표시됩니다. 권한 추가가 필요하면 개발팀에 요청하세요.

**Q: 데이터가 0으로 나와요.**
A: AI 기능(채팅, 브리핑 등)을 한 번도 사용하지 않은 상태입니다. 사용 후 데이터가 쌓입니다.

**Q: 비용은 실제 청구 금액인가요?**
A: 예상 비용이며, 기본 Sonnet 단가 기준입니다. 실제 청구 금액은 Anthropic 콘솔에서 확인하세요.

**Q: 데이터는 어디에 저장되나요?**
A: Google Sheets의 `TokenUsage` 시트에 저장됩니다. 요청/응답 본문은 기록하지 않으며, 호출 메타데이터만 수집합니다.
