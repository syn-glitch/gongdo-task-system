## 개발 목적

### 배경

공도 업무 관리 시스템은 Claude API를 활용하여 채팅(주디), 모닝 브리핑, 업무 추출, 메모 분석 등 다양한 AI 기능을 제공한다. 그러나 **API 호출량과 비용을 추적할 수단이 없어**, 아래 문제가 발생했다:

1. **비용 불투명** — 월간 API 비용이 얼마인지 파악 불가
2. **사용 패턴 미파악** — 어떤 기능이 토큰을 가장 많이 소모하는지 모름
3. **이상 감지 불가** — 비정상적 API 호출 급증을 탐지할 방법 없음

### 해결 방안

모든 Claude API 호출을 **공통 래퍼(`callClaudeAPI()`)**로 감싸 사용량을 자동 기록하고, **Judy Workspace 내 전용 탭**으로 시각화한다.

### 핵심 설계 원칙

| 원칙 | 설명 |
|------|------|
| **Fail-Safe 로깅** | 로깅 실패해도 AI 응답은 반드시 반환 (try-catch 보호) |
| **권한 제어** | 백엔드에서 허용 사용자만 통계 조회 가능 |
| **프라이버시** | 요청/응답 본문은 기록하지 않음 (메타데이터만 수집) |
| **제로 임팩트** | 기존 AI 기능의 성능·안정성에 영향 없음 |

## 아키텍처 개요

```
┌─────────────────────────────────────────────────────┐
│                  Judy Workspace (HTML)               │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌────────┐ │
│  │ 내 노트  │  │ 내 업무  │  │   ...   │  │📈 토큰 │ │
│  └─────────┘  └─────────┘  └─────────┘  └────┬───┘ │
└───────────────────────────────────────────────┼─────┘
                                                │
                    google.script.run            │
                                                ▼
┌─────────────────────────────────────────────────────┐
│              Google Apps Script (Backend)             │
│                                                      │
│  ai_token_logger.gs                                  │
│  ├── callClaudeAPI()     ← 공통 래퍼                  │
│  ├── logTokenUsage()     ← 시트 기록                  │
│  ├── getTokenUsageStats() ← 통계 조회 API            │
│  └── ensureTokenUsageSheet() ← 시트 자동 생성         │
│                                                      │
│  적용 파일:                                           │
│  ├── ai_chat.gs          (2곳)                       │
│  ├── ai_report.gs        (2곳)                       │
│  ├── ai_briefing.gs      (1곳)                       │
│  └── ai_task_parser.gs   (5곳) ← 총 10개 호출 지점    │
└──────────────────────────┬───────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────┐
│              Google Sheets (TokenUsage 시트)          │
│  timestamp | date | functionName | userName |        │
│  inputTokens | outputTokens | totalTokens | model    │
└─────────────────────────────────────────────────────┘
```

## 접근 권한

현재 토큰 대시보드 조회가 허용된 사용자:

- **송용남** (대표)
- **정혜림**

권한 변경은 `ai_token_logger.gs`의 `ALLOWED_TOKEN_VIEWERS` 배열과 `judy_workspace.html`의 `TOKEN_ALLOWED_USERS` 배열을 수정해야 한다.
