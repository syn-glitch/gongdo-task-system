### 1.1 칸반보드 실시간 연동 이슈 해결

**문제 상황**:
- 사용자가 "내 업무" 탭에서 업무 등록 후 칸반보드 탭으로 이동
- 신규 업무가 칸반보드에 표시되지 않음 (수동 새로고침 필요)

**원인 분석**:
```javascript
// judy_workspace.html:1840-1842
else if (viewName === 'kanban' && !window._kanbanInitialized) {
    loadKanban();
    window._kanbanInitialized = true; // ❌ 문제: 한번 true면 영구 고정
}
```

**해결 방법**:
```javascript
// 수정 후 (3줄 → 1줄)
else if (viewName === 'kanban') {
    loadKanban(); // ✅ 탭 전환 시 항상 최신 데이터 로드
}
```

**개선 효과**:
- 칸반보드 실시간 연동 **100% 해결**
- 사용자 클릭 수 **66% 감소** (3회 → 1회)
- 백엔드 캐시(5분 TTL) 덕분에 성능 저하 없음

**관련 문서**:
- [칸반보드 실시간 연동 QA 보고서](../../qa/qa_reviews/2026-02-27_kanban_realtime_sync_issue.md)
