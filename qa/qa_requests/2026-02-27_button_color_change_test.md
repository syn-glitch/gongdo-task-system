# QA 검수 요청서 (테스트용)

**작성일**: 2026-02-27
**작성자**: 자비스 (PO) / 알렉스 (Tech Lead)
**기능명**: 메인 화면 '업무 등록' 버튼 색상 및 로직 변경 테스트

## 1. 변경 요약
- **목적**: 핑퐁(Ping-pong) QA 워크플로우를 테스트하기 위한 가상의 기능 변경.
- **주요 내용**:
  1. 메인 화면의 '업무 등록' 버튼 색상을 기본 파란색에서 **빨간색(Red)**으로 변경. (UX)
  2. 버튼 클릭 시 임의의 3자리 난수를 생성하여 콘솔에 로그로 출력. (로직)
  3. 콘솔에 출력할 때 개인 식별 정보(전화번호 형식의 더미 데이터)도 함께 출력해 봄. (보안)

## 2. 변경 파일 (가상)
- `judy_workspace_final_fix.html` (버튼 스타일 및 클릭 이벤트 추가)

## 3. 테스트 시나리오
1. 앱 진입 시 '업무 등록' 버튼이 빨간색인지 확인.
2. 버튼 클릭 시 에러 없이 "랜덤 난수: XXX"가 출력되는지 확인.
